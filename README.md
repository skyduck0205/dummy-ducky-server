# Dummy Ducky Server
Mock API server for front-end development.

## Environment
- node v8.9
- npm v5.5

### For nvm
```shell
nvm use
```

## Run server
```shell
npm install
npm start
```
The server will serve on port 8888. You can connect to `http://localhost:8888` to manage your mock APIs.

### Backend debug logs
```shell
DEBUG=dummy-ducky-server:* npm start
```
